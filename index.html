Installation

Follow the steps below to get MyBot up and running on your Discord server:

  1. Invite MyBot to your server using the following link: Invite Link
  2. Make sure the bot has the necessary permissions.
  3. Use the command !setup to configure the bot.

Bot Commands

Here are some of the most common commands available with MyBot:

!help

Displays a list of all available commands.

!ban [user]

Bans a specified user from the server.

!kick [user]

Kicks a specified user from the server.
