Welcome to RadiationGuard Documentation!

Your trusted companion for keeping your Discord server safe, secure, and fun. Learn how to install and use the bot below!

Installation

Follow these steps to get started with RadiationGuard:

  1. Invite RadiationGuard to your server using the link below: Invite Link
  2. Ensure the bot has the required permissions to function properly.
  3. Run the setup command ?setup to configure your bot.

Bot Commands

RadiationGuard offers a variety of useful commands for managing your server:

?help

Shows a list of all available commands and their descriptions.

?ban [user]

Bans the specified user from the server.

?kick [user]

Kicks the specified user from the server.

?mute [user]

Mutes the specified user in voice and text channels.

?unmute [user]

Unmutes the specified user.

?roles

Displays all the roles in the server.
