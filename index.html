Privacy Policy

Last Updated: 12 December 2024

1. Introduction

This Privacy Policy outlines how RadiationGuard collects, uses, and protects data from users on Discord servers where the feature is enabled. By using this feature, you agree to the terms described in this policy.

2. Information We Collect

  • Message Content: Messages sent in monitored channels may be analyzed for moderation or analytics purposes.
  • User Information: Limited metadata, including user IDs, timestamps, and server IDs, may be collected to associate messages with users and servers.
  • Flagged Content: Messages flagged for spam, offensive language, or other violations.

3. How We Use Collected Data

  • Moderation: Identifying and mitigating harmful or unwanted behavior.
  • Analytics: Providing server administrators with insights into activity trends.
  • Compliance: Enforcing server-specific rules and Discordâ€™s Terms of Service.

4. Data Storage and Security

  • Storage: Collected data is stored on secure servers for a limited duration as defined by the server administrator.
  • Security Measures: Encryption and access controls are implemented to protect data from unauthorized access.

5. Data Sharing

We do not share collected data with third parties unless required by law or explicitly authorized by the server owner.

6. User Rights

  • Request the removal of their data where applicable.
  • Request access to the data stored about them.

7. Administrator Control

  • Enable or disable RadiationGuard for their server.
  • Manage data retention policies according to server requirements.

8. Limitations

  • RadiationGuard does not monitor private messages (DMs) unless explicitly enabled by both parties.
  • Data is not retained indefinitely without administrative control.
